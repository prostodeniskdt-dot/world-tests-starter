## ТЕСТ 1 (Простой). **«Пять колонн пузырьков: базовая сборка»**

**Вопрос 1.** Что точнее всего отражает смысл «зоны комфорта» для газа в жидкости? *(Best paraphrase / Single select)*
A) Лучшее действие — поднять давление, чтобы ускорить карбонизацию, для напитка, при заданных условиях.
B) Лучшее действие — выровнять пять факторов, чтобы растворять газ стабильнее, для напитка, при заданных условиях.
C) Лучшее действие — увеличить сахар, чтобы усилить пузырьки, для напитка, при заданных условиях.
D) Лучшее действие — снизить кислотность, чтобы смягчить вкус, для напитка, при заданных условиях.
E) Лучшее действие — поднять температуру, чтобы сделать вкус ярче, для напитка, при заданных условиях.
F) Лучшее действие — поднять крепость, чтобы удержать газ дольше, для напитка, при заданных условиях.
**Правильный ответ: B**
**Пояснение:** «Зона комфорта» — это баланс нескольких опор, а не один рычаг. Варианты про один фактор путают цель устойчивости со скоростью.
