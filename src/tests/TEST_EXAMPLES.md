# Примеры правильно оформленных тестов

## Пример 1: Полный тест с разными механиками

```
ТЕСТ 1 — «Основы коктейлей»

Вопрос 1. Какой главный ингредиент в Мохито? (Single select)
A) Ром
B) Водка
C) Джин
D) Текила
Правильный ответ: A
Пояснение: Мохито традиционно готовится на основе белого рома.

---

Вопрос 2. Какие ингредиенты обязательны в классическом Маргарите? (Multiple select) (Верных: 3)
A) Текила
B) Лаймовый сок
C) Апельсиновый ликер
D) Сахарный сироп
E) Содовая
F) Биттер
Правильный ответ: A, B, C
Пояснение: Маргарита состоит из текилы, лаймового сока и апельсинового ликера (Triple Sec или Cointreau).

---

Задание 3. Соотнесите коктейль и его базовый спирт. (Matching)
Слева:
1. Old Fashioned
2. Negroni
3. Moscow Mule
4. Daiquiri
Справа:
A) Ром
B) Водка
C) Джин
D) Виски
Ответ: 1–D, 2–C, 3–B, 4–A
Пояснение: Каждый классический коктейль имеет традиционную базу.

---

Задание 4. Расставьте этапы приготовления Dry Martini. (Ordering)
Элементы:
A) Охладить бокал льдом
B) Смешать джин и вермут в миксерном стакане
C) Размешать с льдом 20-30 секунд
D) Процедить в охлажденный бокал
Ответ: A → B → C → D
Пояснение: Правильная последовательность обеспечивает оптимальную температуру и разбавление.

---

Вопрос 5. Утверждение: «Шейкер всегда дает лучший результат, чем смешивание». (True/False + Reason)
   1. Верно/Неверно: Неверно
   2. Выберите причину:
A) Смешивание предпочтительно для спиртовых коктейлей без соков
B) Шейкер всегда используется для любых коктейлей
C) Смешивание невозможно выполнить правильно
D) Температура не важна при приготовлении коктейлей
Ответ: Неверно; причина: A
Пояснение: Спиртовые коктейли без соков (как Martini, Manhattan) лучше размешивать, чтобы избежать лишней аэрации.

---

Ключ (Тест 1):
1A; 2A,B,C; 3(1D2C3B4A); 4(A→B→C→D); 5Неверно/A.
```

## Пример 2: Тест с продвинутыми механиками

```
ТЕСТ 2 (Средний) — «Техники баланса»

Задание 1. Разнесите компоненты по категориям. (Grouping/Classification)
Категории: «Добавляют сладость», «Добавляют кислотность»
Элементы:
A) Сахарный сироп
B) Лаймовый сок
C) Мед
D) Лимонный сок
E) Агавовый сироп
F) Грейпфрутовый сок
Ответ: «Добавляют сладость»: A, C, E; «Добавляют кислотность»: B, D, F
Пояснение: Баланс сладости и кислотности — основа многих коктейлей.

---

Задание 2. Заполните пропуски. (Dropdown cloze)
Текст: «Соотношение спирт:цитрус:сироп в классическом Sour составляет [1: ___], что создает идеальный баланс».
Варианты для [1]:
A) 2:1:1
B) 1:1:1
C) 3:2:1
D) 1:2:1
Ответ: [1]=A
Пояснение: Формула 2:1:1 (2 части спирта, 1 часть цитруса, 1 часть сиропа) — классическая основа семейства Sour.

---

Задание 3. Найдите 2 ошибки в утверждении. (Select errors)
Текст:
[1] «Лед в коктейле только охлаждает напиток».
[2] «Разбавление водой от льда является нежелательным эффектом».
[3] «Крупный лед тает медленнее и меньше разбавляет напиток».
[4] «Встряхивание в шейкере обязательно для всех коктейлей».
[5] «Биттеры добавляют малыми дозами для комплексности».
Ответ: 2, 4
Пояснение: Разбавление — это часть процесса, а не ошибка. Не все коктейли требуют шейкера.

---

Задание 4. Двухступенчатое: выбор техники для газированного коктейля. (Two-step / branching)
Задание 4 (Часть 1). Как правильно добавить содовую в коктейль?
A) Влить быстро сразу после шейкера
B) Долить аккуратно последним ингредиентом без перемешивания
C) Смешать в шейкере со всеми ингредиентами
D) Добавить в миксерный стакан перед размешиванием
Задание 4 (Часть 2). Выберите объяснение:
A) Содовая теряет газ при активном перемешивании, поэтому её добавляют последней
B) Содовая требует интенсивного встряхивания для лучшего смешивания
C) Содовая должна быть добавлена первой для правильной температуры
D) Содовая не влияет на текстуру и может добавляться как угодно
Ответ: Ч1=B; Ч2=A
Пояснение: Содовую добавляют последней легкими движениями, чтобы сохранить карбонизацию.

---

Задание 5. Для каждой ошибки подберите последствие. (Grid / matrix)
Объекты:
1. Слишком много льда в шейкере
2. Недостаточное встряхивание
3. Слишком долгое размешивание
Характеристики:
A) Чрезмерное разбавление напитка
B) Недостаточное охлаждение и слабое смешивание
C) Излишнее разбавление при размешивании
D) Правильный баланс температуры
Ответ: 1–A; 2–B; 3–C
Пояснение: Каждая ошибка техники имеет свои последствия для конечного результата.

---

Ключ (Тест 2):
1(A,C,E / B,D,F); 2[1A]; 3(2,4); 4(Ч1=B;Ч2=A); 5(1A2B3C).
```

## Пример 3: Тест с ситуационными вопросами

```
ТЕСТ 3 (Сложный) — «Работа за стойкой»

Задание 1. Ситуация: гость просит «что-то освежающее, но не слишком сладкое». Что предложить? (Scenario-based mini-case)
A) Mojito с увеличенной дозой мяты и меньшим количеством сахара
B) Pina Colada с кокосовым сиропом
C) Long Island Iced Tea
D) Mudslide с шоколадным ликером
Правильный ответ: A
Пояснение: Mojito с коррекцией сладости идеально подходит под запрос: освежающий за счет мяты и лайма, контролируемая сладость.

---

Задание 2. Ситуация: закончился Triple Sec для Margarita. Чем заменить? (Scenario-based mini-case)
A) Cointreau или Grand Marnier (другие апельсиновые ликеры)
B) Амаретто (миндальный ликер)
C) Campari (горький ликер)
D) Bailey's (сливочный ликер)
Правильный ответ: A
Пояснение: Cointreau и Grand Marnier — это премиальные апельсиновые ликеры, которые отлично заменяют Triple Sec в Маргарите.

---

Задание 3. Ситуация: гость жалуется, что коктейль «слишком крепкий». Как исправить не переделывая? (Scenario-based mini-case)
A) Добавить немного сока или содовой для разбавления
B) Добавить больше крепкого алкоголя для баланса
C) Удалить часть напитка и налить новый
D) Ничего не делать, это нормальная крепость
Правильный ответ: A
Пояснение: Добавление сока или содовой (в зависимости от типа коктейля) — самый простой способ снизить крепость уже готового напитка.

---

Ключ (Тест 3):
1A; 2A; 3A.
```

## Важные замечания при создании тестов

### 1. Пояснения должны быть информативными

**Хорошо:**
```
Пояснение: Мохито традиционно готовится на основе белого рома. Использование других спиртов создаст другой коктейль с другим названием.
```

**Плохо:**
```
Пояснение: Правильный ответ A.
```

### 2. Варианты ответов должны быть примерно одной длины

**Хорошо:**
```
A) Белый ром
B) Темный ром
C) Золотой ром
D) Пряный ром
```

**Плохо:**
```
A) Белый ром
B) Темный ром, который выдерживается в дубовых бочках минимум 3 года
C) Ром
D) Любой
```

### 3. Избегайте абсолютных слов в неправильных ответах

Слова типа "всегда", "никогда", "обязательно" часто делают неправильные ответы слишком очевидными.

**Хорошо:**
```
B) Шейкер обычно используется для коктейлей с соком
```

**Плохо:**
```
B) Шейкер ВСЕГДА используется для ЛЮБЫХ коктейлей
```

### 4. Для matching убедитесь, что соответствия однозначны

**Хорошо:**
```
Слева:
1. Водка (40%)
2. Ликер (20%)
3. Биттер (45%)
4. Вермут (18%)

Справа:
A) Креплёный винный напиток
B) Крепкий нейтральный спирт
C) Сладкий ароматизированный спирт
D) Концентрированная настойка
```

**Плохо (неоднозначно):**
```
Слева:
1. Спирт
2. Спирт крепкий
3. Спирт
4. Спиртное

Справа:
A) Напиток
B) Алкоголь
C) Жидкость
D) Спирт
```

## Чек-лист перед добавлением теста

- [ ] Заголовок теста в правильном формате
- [ ] Все вопросы пронумерованы последовательно (1, 2, 3...)
- [ ] Указан тип механики для каждого вопроса
- [ ] Все варианты ответов начинаются с A), B), C), D)...
- [ ] Есть правильный ответ для каждого вопроса
- [ ] Есть пояснение для каждого вопроса
- [ ] Ключ ответов в конце теста начинается с "Ключ (Тест X):"
- [ ] Все ответы в ключе соответствуют номерам вопросов
- [ ] Для matching указаны "Слева:" и "Справа:"
- [ ] Для ordering/grouping указаны "Элементы:"
- [ ] Для grouping указаны "Категории:"
- [ ] Для cloze указан "Текст:" с пропусками [1: ___]
- [ ] Для matrix указаны "Объекты:" и "Характеристики:"
- [ ] Для two-step есть "(Часть 1)" и "(Часть 2)"
- [ ] Нет опечаток в тексте вопросов и ответов
- [ ] Все символы (тире, стрелки, кавычки) в правильном формате
